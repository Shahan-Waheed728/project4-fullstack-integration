# Project 4 — Frontend & Backend Integration

## Overview

A full stack web application integrating frontend portfolio with backend API and MongoDB database as part of DecodeLabs Full Stack Development Internship 2026.

---

## Technologies Used

### Frontend

* HTML5 (Semantic markup)
* CSS3 (Flexbox, Grid, Variables)
* JavaScript (Fetch API, Async/Await)

### Backend

* Node.js v22
* Express.js
* MongoDB Atlas
* Mongoose
* CORS
* dotenv

---

## How to Run

### Backend

```bash
cd backend
npm install
npm run dev
```

### Frontend

Open `frontend/index.html` with Live Server.

---

## Features

* Dynamic users dashboard
* Add users via form — saves to MongoDB
* Delete users instantly
* Real time data from backend
* W3C Validated HTML & CSS

---

## API Endpoints

| Method | Endpoint     | Description     |
| ------ | ------------ | --------------- |
| GET    | `/users`     | Get all users   |
| GET    | `/users/:id` | Get single user |
| POST   | `/users`     | Create new user |
| PUT    | `/users/:id` | Update user     |
| DELETE | `/users/:id` | Delete user     |
